Change Log
----------

This represents a summary of additions, changes and removals for the 0.5.0 release of the dotNetRDF Toolkit
for Windows/Mono detailed by Tool.

Note: While some tools may have no changes all tools benefit from being upgraded to the latest release of the
dotNetRDF core library which typically incorporates bug fixes and performance improvements so you should still
upgrade to these latest versions.

rdfConvert
- Improved -verbose mode to show progress monitoring

rdfEditor (Not available under Mono)
- Default for Use UTF-8 BOM is now false
 
rdfOptStats
   
rdfQuery

rdfServer
- New command line tool for running a basic SPARQL Query and Update server

rdfServerGUI
- New GUI tool to make it easy to create rdfServer instances for testing

rdfSqlStorage
 
rdfWebDeploy

soh
