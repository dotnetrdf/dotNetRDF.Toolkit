dotNetRDF Store
===============

Management Tool for managing dotNetRDF stores which are a database format that can be used with
Microsoft SQL Server, MySQL Server and Virtual Databases in Virtuoso Universal Server.

The tool allows you to perform admin tasks on databases such as creating and deleting stores as well
as viewing and editing their contents.

Note - This Tool is a work in progress and not all features are fully implemented or debugged yet.

Version 0.1.1 Alpha

Robert Vesse 2009
rvesse@vdesign-studios.com

Usage
-----

Run the dotNetRDFStore.exe program and then connect to your chosen database.  Use the Database Tools tab
of the Store Manager form to create/delete a Store in that database as required.
